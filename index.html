National Parks and Campgrounds Finder

This is an app for outdoor enthusiasts, National Park explorers, and anyone seeking a camping adventure. I created this web app to learn about using multiple APIs in a project. It queries the National Parks API to find parks located in or around a state selected by the user. When the list of parks is displayed, the user can click on the "show campgrounds" button. It will query the campgrounds API endpoint and display all the available campgrounds for that park. I built this app because my fiance and I spent the summer of 2020 exploring National Parks and fell in love with the outdoors! Feel free to try it out to help plan your next trip!

Technologies used:

  • HTML
  • CSS
  • JavaScript
  • jQuery
  • RESTful APIs
