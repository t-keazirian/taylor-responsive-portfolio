I built Live With Intent as a goal tracking web app. This app is for anyone who loves to keep track of their goals in a detailed format. I designed both the front-end and the back-end, and RESTful APIs. The user can create new goals, add notes, edit their current goals, and keep track of their progress.

National Parks and Campgrounds Finder

This is a web app I created to learn about using multiple APIs in a project. It searches the National Parks API to find parks based in a state selected by the user. When the list of parks are displayed, click on the "show campgrounds" button. It will query the campgrounds endpoint and display all the available campgrounds for that park. This web app is for anyone who loves to explore the outdoors!

Technologies used:

  • HTML
  • CSS
  • JavaScript
  • jQuery
  • RESTful API Endpoints

repository | live app
