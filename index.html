Live With Intent

I built Live With Intent as a goal-tracking web app. I designed both the front-end and the back-end - this was my most comprehensive project at Thinkful. This app is for anyone who loves to keep track of their goals in a detailed format. The user can create new goals, add notes, edit their current goals, and keep track of their progress. National Parks and Campgrounds Finder

This is an app for outdoor enthusiasts, National Park explorers, and anyone seeking a camping adventure. I created this web app to learn about using multiple APIs in a project. It queries the National Parks API to find parks located in or around a state selected by the user. When the list of parks is displayed, the user can click on the "show campgrounds" button. It will query the campgrounds API endpoint and display all the available campgrounds for that park. I built this app because I spent the summer of 2020 exploring National Parks in Montana with my fiance and fell in love! Feel free to try it out to help plan your next trip!

Technologies used:

  • HTML
  • CSS
  • JavaScript
  • jQuery
  • RESTful APIs
